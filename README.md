# 🚀 Tablón Hogar Inteligente

¡Bienvenido al centro de control y organización de nuestro hogar! Esta es una aplicación desarrollada en Python diseñada para automatizar tareas, motivar a la familia mediante la gamificación y centralizar el control de dispositivos inteligentes.

---

## 📌 Características Principales

**📈 Sistema de Puntos y Equidad:** Registro de tareas del hogar con valores dinámicos basados en la dificultad (1 al 10).
**🏆 Rincón de Premios (Gamificación):** Perfil especial para los niños con barras de progreso visuales para alcanzar metas y recompensas a largo plazo.

##PROXIMAMENTE😃
**📺 Control de Fire TV Integrado:** Botones de acceso rápido desde el celular para lanzar aplicaciones (Netflix, YouTube, Prime Video) a través de la red local (ADB/Python).
 **🎙️ Automatización por Voz (Próximamente):** Control de luces y ambientes mediante comandos de voz nativos.

---

## 🛠️ Tecnologías Utilizadas

* **Lenguaje:** Python 3.x
* **Control de Versiones:** Git y GitHub
* **Entorno de Desarrollo:** Visual Studio Code

---

## 📱 Vista del Proyecto (Lógica de Desarrollo)

El proyecto está estructurado para separar la lógica de control con la interfaz de usuario:

1. **Backend (Python):** Se encarga de procesar los inputs de datos, calcular los porcentajes de la barra de progreso y enviar los comandos de red al Fire TV.
2. **Frontend:** Interfaz limpia y táctil diseñada para el uso diario en dispositivos móviles.
